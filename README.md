# Student Registration System

A simple front-end Student Registration System.

## Features
- Register students (name, ID, email, contact)
- Client-side validation
- View saved student records (localStorage)
- Edit and delete records

## How it works
- `index.html` contains the registration form.
- `record.html` shows the registered student table.
- `script.js` stores and manages students in `localStorage`.
- `styles.css` provides styling.

## Run
1. Open `index.html` in a browser.
2. Register students and view records.
